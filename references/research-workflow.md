# 市场调研工作流

## 目标

寻找当前品类中反复出现、适合输入商品的电商视觉语言。不要复制单个竞品，也不要把搜索排名或内容互动误写成真实销量。

## 1. 建立检索词

先从产品图和用户信息提取：

- 中文、英文商品类别。
- 材质、工艺、用途、目标人群。
- 风格、价格感、礼赠或日用场景。
- 同义词、俗称和平台常用商品词。

组合检索词，例如：

```text
<品类> 热卖 商品图
<材质> <品类> 高端 电商
<品类> 场景图 细节图
<category> product photography bestseller
<material> <category> lifestyle product image
```

## 2. 收集当前样本

网络可用时必须实时调研。优先收集：

1. 电商平台搜索结果中的平台热卖或畅销标记。
2. 多个头部店铺、品牌官方商品页。
3. 近期内容平台高互动商品内容。
4. 同材质、同文化语境、同价位的设计案例。

尽量比较 6–12 个有效样本，覆盖至少 3 个不同商家或品牌、2 类来源。无法达到时使用现有有效样本，不虚构数量。

## 3. 记录证据

在 `research.md` 为每个样本记录：

```markdown
### 样本名称
- 日期：
- 平台：
- 链接：
- 证据类型：
- 可见视觉特征：
- 可借鉴原则：
- 不应复制的品牌元素：
```

证据类型仅使用：

- `platform-bestseller-marker`：平台明确展示热卖、畅销等标识。
- `engagement-signal`：内容页有可见互动信号。
- `search-prominence`：搜索结果靠前，但不能等同销量。
- `official-brand-example`：品牌或店铺官方视觉。
- `design-only-reference`：只用于审美和构图参考。

## 4. 推导视觉趋势

横向比较样本，提取重复出现的：

- 背景明暗和材质。
- 商品占画面比例和镜头角度。
- 灯光方向、软硬和反光控制。
- 常见使用场景和道具。
- 细节图聚焦的工艺证据。
- 文案长度、位置和字体气质。
- 主图、氛围图、卖点图之间的分工。

只把多个独立样本反复出现的特征写成“热门视觉趋势”。单个案例只能写成“可选设计参考”。

## 5. 形成视觉简报

将结论写入 `visual-brief.md`：

```markdown
# Visual Brief

## 产品与用户
## 市场证据摘要
## 核心风格与价格感
## 主色、辅色和背景材质
## 光线、镜头和构图
## 场景与道具
## 六张图的销售任务
## 全局禁用项
## 证据限制
```

## 禁止事项

- 不声称“销量最高”“转化最好”，除非有直接、可信数据。
- 不虚构销量、评价、市场份额或消费者结论。
- 不复制竞品 Logo、包装、专属纹样、品牌文案或受保护角色。
- 不因流行趋势改变真实商品结构、功能或材质。
- 不把广告软文当独立证据。

## 调研降级

若网络不可用、页面无法访问或样本不足：

1. 使用保守的品类启发式：清晰主图、真实材质、可信使用场景、克制道具、短文案。
2. 在 `research.md` 写明失败原因和尝试过的检索方向。
3. 在 `visual-brief.md` 和最终回复标注：`未完成实时市场验证`。
4. 继续生成，不因非关键调研阻塞整个任务。
